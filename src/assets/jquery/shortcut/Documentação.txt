    Segue um script muito bacana para criar teclas de atalho em suas páginas HTML,
    permitindo uma nova acessibilidade de suas páginas utilizando o teclado.
     
    Existe duas funções, uma para criar a tecla de atalho e outra para removê-la.
     
    Ao pressionar a sequencia de teclas CTRL+X por exemplo, será exibido no caso desta página
    um alerta dizendo o que foi pressionado. Mas você pode criar vários tipos de exemplos
    como abrir um menu, colocar foco em um objeto, aí vai.
     
    shortcut.add()
    /*
    Exemplo para criar a tecla de atalho
    */
    shortcut.add("Ctrl+X",function() {
    	alert("Foi pressionado a sequencia de teclas CTRL+X!");
    });
     
     
    shortcut.remove()
    /*
    Exemplo para remover a tecla criada
    */
    shortcut.remove("Ctrl+X");
     
    As teclas válidas são:
     
    Todas as letra e números - abc...xyz,01..89
